# Samsara-Adventures-Preview

Tech Stack: Kotlin, Jetpack Compose, Retrofit, Coil, Carousel Slider, Permission Handler, Razorpay
Backend: Node.js, Express.js, MongoDB, JWT, Nodemailer, Multer, Cloudinary
Authentication: Email/Password, Google Sign-In

Project Overview
Samsara Adventures is a modern and intuitive trip and event booking application that allows users to explore curated travel packages and upcoming events across India. The app focuses on simplicity, customization, and secure online booking. Users can browse packages, filter by location and category, customize their trips, and complete payments seamlessly through a responsive UI built entirely with Jetpack Compose.

Key Features

User Authentication
- Secure email and password login/signup using backend JWT authentication.
- Google Sign-In integration for quick onboarding.
- Users can update their profile and upload/change profile images (stored via Cloudinary).

Explore Trips & Events
- Home screen with banners and featured packages using a carousel slider.
- Browse multiple categories such as adventure trips, family trips, solo trips, weekend events, etc.
- Smooth image loading using Coil.

Advanced Filtering & Search
Users can filter packages based on:
- State/Location
- Category (Adventure, Family Trip, Solo Trip, Weekend Event, etc.)
- Price Range
- Trip Type
- Custom Date Range
This improves discoverability and provides a personalized experience.

Custom Package Booking
- Users can create custom itineraries, select destinations, and add personalized activities.
- Custom booking requests are validated and sent to the backend.

Secure Online Payments
- Razorpay Payment Gateway integration for fast and secure bookings.
- Supports UPI, cards, wallets, and net banking.
- Real-time update of payment status.

Profile Management
- Users can update their personal details and mobile number.
- Profile photo upload using Multer + Cloudinary.
- View booking history and booking status.

Permissions & Device Handling
- Integrated Permission Handler for location, storage, and camera permissions.

Modern UI with Jetpack Compose
- Fully built with Jetpack Compose using smooth animations and material design components.
- Clean architecture using MVVM, StateFlow, and repository pattern.

Backend Features (Node.js, Express)
RESTful API Architecture
- Modular routing and controller structure.
- All APIs secured with JWT authentication and refresh token support.

MongoDB Database
- Stores user profiles, trips, bookings, events, and custom booking requests.
- Mongoose schema validation ensures consistency.

Media Uploads
- Image upload handled with Multer.
- Images stored on Cloudinary with CDN optimization.

Email Communication
- Nodemailer used for booking confirmation emails, OTP/verification emails, and custom trip request notifications.

Additional Highlights
- Smooth animations and transitions.
- Proper loading, error handling, and retry states.
- Clean Architecture and modular codebase.
- Optimized Retrofit API calls using Kotlin Coroutines.
- Network state handling with sealed classes.
- Light/Dark theme support (optional).

Project Impact
Samsara Adventures delivers a seamless and modern booking experience while providing businesses a scalable platform to showcase and promote trips and events. The project demonstrates strong skills in full-stack mobile development with clean architecture, secure authentication, and high-quality UI/UX.
